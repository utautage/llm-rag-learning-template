# 私の学習支援システム

## 実装した機能

### 基本機能

- LLMによる質問応答
- RAGによる文書検索
- オントロジーによる概念管理

### 独自機能

- ユーザが入力した英文に対して、LLMが添削したり、自然な言い回しを提案したりする。
- 会話の舞台に関連する文化や背景をRAGで検索し、雑談として教えてくれる。
- 「挨拶」や「食べ物の注文」といった概念をオントロジーで管理し、マスター度合いに応じて物語を進む。

## 工夫したポイント

1. **ゲーム要素**：『[Risdom](https://ris-dom.com/)』というゲームでは、英語を学習しないとリズムバトルをクリアできない。そのようなゲーム的要素を、LLM・RAG・オントロジーの技術と組み合わせて取り入れてみようと考えた。

## 使い方

1. `my-learning-system.html`をブラウザで開く
2. 初期状態では「注文する」ボタンがロックされている。「店員に挨拶する」をクリックすると、「greeting」の概念をマスターし、ボタンのロックが解除される。
3. ページ下の入力欄からAIパートナーに自由に話しかけたり、添削を依頼したりできる。
4. シーン開始時には、AIが豆知識を教えてくれる。

## 今後の改善予定

- 現在は「挨拶」の概念をマスターしたところで終わりである。今後、物語や学習する概念を拡張してシステムを作りこみたい。
